Tutorial rapido

Ti mostriamo come usare le funzionalità principali in meno di un minuto.

1) Importa o crea una storia

Dalla home premi il pulsante + per aprire la sezione editor. Puoi incollare un JSON o caricare un file .json.

2) Leggi e scegli

Apri una storia dall'elenco e usa le scelte disponibili per avanzare tra le pagine. Se arrivi a una fine puoi ricominciare in qualsiasi momento.

3) Cambia lingua e sincronizza

In alto a destra puoi cambiare lingua (IT/EN). Le storie importate da GitHub vengono sincronizzate automaticamente.

4) Riapri il tutorial quando vuoi

Dopo la prima chiusura troverai il pulsante ? nella schermata principale per rivedere questa guida.
